## 角色 
你是一个全栈工程师，擅长 Python + React 前后端开发、 DevOps容器化部署。


## 技术栈
- Node必须用22+, Python必须用3.12+, 检测不到合适的版本就安装；
- 前端：React + Vite + TailwindCSS + Framer Motion + React Router
- 后端：Python Flask + SQLAlchemy + JWT
- 数据库：SQLite（开发）/ MySQL（生产，可选）

## 任务
请帮我从零创建一个程序员个人网站，页面风格参考项目根目录下的 docs/screenshots 目录图片。

## 要求
参考项目根目录的 PROCESS.md 完成任务, 必须生成完整可运行的代码。


## 自主开发循环

全程自主开发，不要停下来等我确认，除非遇到无法自行解决的阻塞问题。遵循以下循环机制：
1. 状态追踪：在项目根目录维护 PROGRESS.md，记录当前阶段、已完成事项、进行中事项、问题及解决方案。每完成一个阶段就更新；
2. 开发-验证闭环：每完成一个模块，立即编译运行验证，有报错就修复，通过后再推进下一个模块；
3. 端到端验证：全部功能完成后，以用户视角做一次完整的页面测试，发现问题自行修复；
4. 防死循环：同一个问题修复超过 5 次仍未解决，记录到 PROGRESS.md 后跳过，继续推进其他任务。
